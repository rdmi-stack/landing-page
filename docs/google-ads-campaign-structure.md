# Google Ads Campaign Structure for RDMI Landing Pages

This document maps ad groups to keyword themes, landing page URLs, and responsive search ad copy aligned with the current website content.

## Account Structure Recommendation

- Use tightly themed ad groups, not classic SKAGs.
- Send broad service intent to the homepage only when a tighter keyword landing page does not exist.
- Send high-intent commercial keywords to the closest `/kw/` landing page for stronger ad relevance and landing page experience.
- Keep match types focused on phrase and exact for launch, then expand based on search term reports.

## Launch Notes

- Start each campaign with phrase and exact only.
- Use one landing page per strong keyword theme.
- Review search terms every 3-4 days for negatives and new exact-match opportunities.
- Move high-converting queries from homepage campaigns to dedicated keyword-page campaigns.
- Optimize for qualified lead submissions, not just form volume.
